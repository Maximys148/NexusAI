**Nexus** — это современное десктопное приложение для удобной работы с различными нейросетями в одном месте.

---

## 🌟 Основные возможности

-  **Современный UI**: Темная тема на базе **AtlantaFX**.
-  **Автообновление**: Система профессионального автообновления прямо из GitHub Releases.

---

## 🛠 Установка и запуск

1. Перейдите в раздел [Releases](https://github.com/Maximys148/Nexus/releases/latest)
2. Скачайте архив `Nexuszip`
3. Распакуйте в удобное место
4. Запустите `Nexus.exe`
