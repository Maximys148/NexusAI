#  Nexus  — Твой персональный хаб нейросетей

**Nexus AI** — это современное десктопное приложение для удобной работы с различными нейросетями в одном месте.

---

## 🌟 Основные возможности

-  **Современный UI**: Темная тема на базе **AtlantaFX**.
-  **Автообновление**: Система профессионального автообновления прямо из GitHub Releases.

---

## 🛠 Установка и запуск

1. Перейдите в раздел [Releases](https://github.com).
2. Скачайте последний архив `Nexus.zip`.
3. Распакуйте в удобное место.
4. Запустите `NexusAI.exe`.
