# 🌌 Nexus AI — Твой персональный хаб нейросетей

![Java](https://shields.io)
![Spring](https://shields.io)
![JavaFX](https://shields.io)

**Nexus AI** — это современное десктопное приложение для удобной работы с различными нейросетями в одном месте. Никаких браузерных вкладок — только мощный и быстрый нативный интерфейс.

---

## ✨ Основные возможности

- 🤖 **Единый чат**: Поддержка текстовых моделей (в разработке).
- 🎨 **Современный UI**: Темная тема на базе **AtlantaFX (Nord)**.
- 🚀 **Smart Update**: Система профессионального автообновления прямо из GitHub Releases.
- ⚡ **Spring Boot Backend**: Надежная архитектура и высокая производительность.

---

## 🛠 Установка и запуск

1. Перейдите в раздел [Releases](https://github.com).
2. Скачайте последний архив `NexusAI_vX.X.X.zip`.
3. Распакуйте в удобное место.
4. Запустите `NexusAI.exe`.
